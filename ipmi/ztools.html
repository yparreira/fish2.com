Some tools. Well, for now, tool.

A small python program (>50% inline comments, 2.5k gzip'd) that sends a single packet to a BMC and mulls over the response. What can you do with only a single packet, one might ask? 10+ different security tests for IPMI, for starters. Well, for starters and for enders, it's only a packet :) Requires python, a BMC and an open path to UDP port 623 to work. Usage is simply "ipmi-get-auth.py target".

ipmi-get-auth.py / A very small description

I've written a wrapper around good ol' Nmap to look for systems running BMC. The first problem is that... well, it's not always easy knowing if something is a BMC or not, since UDP 623 (what IPMI talks over) is usually blocked (and if it's not that's another problem ;))

And then Nmap, bless it's heart... oh, how I hate that XML output. I initially wrote it in pure plaintext output mode, but as I scanned more and more I junked that for XML... then I remembered why I did it in normal mode in the first place, as my workstation died from memory starvation after a podunk class B was scanned (damn Chrome too - I have 16 GB RAM, I shouldn't run out!)

Anyway, needs work but if someone is desperate I'll consider release.

I've also written a plain data-sucker that does a credentialed configuration suck of a BMC, but I haven't finished the analytics side yet. It will probably, possibly, maybe come out someday....
